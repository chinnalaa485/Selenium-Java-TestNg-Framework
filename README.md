# Selenium Automation Framework

This project is developed using Selenium WebDriver with Java for web application automation testing.

## Tools & Technologies Used
- Selenium WebDriver
- Java
- TestNG
- Maven
- Page Object Model (POM)
- Git & GitHub

## Framework Features
- Page Object Model design pattern
- Reusable utility methods
- TestNG test execution
- Maven dependency management
- Cross-browser testing support
- Easy maintenance and scalability

## Project Structure
- src/main/java -> Page classes & utilities
- src/test/java -> Test scripts
- testng.xml -> Test execution file
- pom.xml -> Maven dependencies

## How to Run
1. Clone the repository
2. Open project in Eclipse/IntelliJ
3. Install Maven dependencies
4. Run testng.xml file
